# 🎬 BotFlix

Aplicação web que recomenda filmes com base no seu humor ou no que você quer assistir.

---

## 🧠 Como funciona
Você descreve seu humor ou preferência, e o app:

1. Envia o texto para um webhook (N8N)  
2. A IA (Gemini) sugere um filme  
3. Os dados são buscados na API do TMDB  
4. O resultado aparece na tela com poster, descrição e nota  

---

## 🛠️ Tecnologias
- HTML, CSS, JavaScript  
- Fetch API  
- N8N  
- IA (Gemini)  
- TMDB API  

---

## ✨ Funcionalidades
- Busca por texto (botão ou Enter)  
- Validação de campo vazio  
- Requisição POST para recomendação  
- Exibição dinâmica de filme  
- Tratamento de erros  
- Layout responsivo
